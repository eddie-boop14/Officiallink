# officiallink.org

**Price. Location. Link.**

Official ticket links and booking pages for 5,500+ attractions across 44 countries. No resellers. No markup. Direct links to official sources.

## Structure

```
/venue/{slug}/index.html   — individual venue pages (5,511)
/france/                   — France city and category hubs
/spain/                    — Spain hubs
/{country}/                — country hub pages
/img/venue/                — venue photos
/index.html                — homepage
```

## Platform rules

- 🎟 Book button: green ✅ verified only. Never guessed.
- 🌐 Site: ✅ verified · ⚠️ prefilled · greyed = missing
- 📍 Maps: ✅ for maps.app.goo.gl pins · ⚠️ for search URLs
- Zero resellers ever.
